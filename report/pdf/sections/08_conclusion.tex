\section{Conclusion}
\label{sec:conclusion}

We built an end-to-end pipeline that turns a single Stage~1 universal adversarial image into a stream of perceptually invisible adversarial photos at PSNR $\approx 25.2$~dB, and evaluated it on $6{,}615$ (clean, adversarial) response pairs against four open \vlm{}s. Splitting the evaluation into two independent axes shows that broad disruption (Output Affected $\approx 66\%$) coexists with narrow payload delivery (Target Injected $\approx 0.227\%$). The two confirmed injections are both URL injections on a code screenshot; the partial and weak cases all sit on screenshots whose semantic class is compatible with the payload. BLIP-2 is fully immune across every test, and a manual transfer test on the strongest case fails on GPT-4o. The course-level takeaway is that the visual channel is real --- it can disturb production \vlm{}s reliably --- but turning that disturbance into a delivered attacker payload is a much harder problem, gated by image semantics, decoder fidelity, and target-model architecture.
